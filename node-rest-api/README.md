# node-api
A simple nodejs api

NodeJS
MongoDB


### Install

```
$ git clone https://github.com/matheusho/node-api.git
$ cd node-api
$ npm install

$ npm start
```

### How to use

You can use the Postman to consume the api
```bash
GET /users
GET /users/:id
POST /users
PUT /users/:id
DELETE /users/:id
```
